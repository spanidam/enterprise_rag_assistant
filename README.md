# Enterprise RAG Productivity Assistant

## Project Overview
This project implements an enterprise-grade Retrieval-Augmented Generation (RAG)
assistant designed to improve productivity by enabling accurate question answering,
document summarization, and knowledge retrieval from internal documents.

## Features
- Question Answering with citations
- Document summarization
- Semantic and keyword-based search
- Action item extraction
- Document comparison
- Hallucination reduction using grounded generation

## Target Users
- Students
- Knowledge workers
- Researchers

## Tech Stack
- Python 3.10+
- LangChain / LlamaIndex
- FAISS / ChromaDB
- Sentence Transformers
- FastAPI
- Streamlit
