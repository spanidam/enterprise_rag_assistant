(i) Deﬁnition 7.4.2
A linear system is said to be completely state-controllable, or controllable
for short, if for each initial state x(t0) and each ﬁnal state x(tf) there is a control
input t →u(t) that transfers the system from x(t0) at time t −o to x(tf) at
time tf.
The above deﬁnition says, in essence, that if a system is controllable we can
achieve any state whatsoever in ﬁnite time starting from an arbitrary initial
state. To check whether a system is controllable we have the following simple
