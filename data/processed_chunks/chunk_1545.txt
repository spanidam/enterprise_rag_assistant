Finding the GCD
""" Find GCD using recursive implementation of Euclid's method """
def gcd(a, b):
if a == b: # Base case
return a
elif a < b: # Recursive case
return gcd(a, b - a)
else:
return gcd(a - b, a)
# Test code
print(gcd(24, 30))
The above code's output is:
6
